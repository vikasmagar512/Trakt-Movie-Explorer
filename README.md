<h1 align="center">Trakt Movie Explorer</h1>

- View Trending, Popular movies 
- Search for movies
- Browse through movie's cast
- See the movies a perticular actor has worked in

Technical Details

- React-redux App 
- No Server involved
- Third Party Authentication
